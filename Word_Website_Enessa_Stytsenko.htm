Simple Personal Web Page (HTML Export from Word)

This document is a simple web page created in Microsoft Word and intended to be saved as a .html file.

To export:
1. Go to File → Save As.
2. Choose the format 'Web Page (.htm or .html)'.
3. Name the file 'index.html'.
4. A folder containing images and styles will be created alongside the HTML file.
